### Portfolio Website

This is my personal portfolio website created as part of the MCA Major Project.
The website showcases my skills, projects, education, and contact details. It is designed to be clean, responsive, and easy to navigate on mobile, tablet, and desktop screens.

### Features

- Home/Hero section with introduction

- About me section

- Skills divided into categories

- Projects

- Education background

- Downloadable resume

- Contact form (fully working) Used formspree to send real time mails.

### Tech Stack Used

- HTML
- CSS
- JS
- Github Pages to deploy the website

### How to Use:

- Download or clone the repository
- Open index.html in any browser
- Make sure all files are inside the same folder
- For the contact form to work, update your Formspree form ID in script.js
